  3. Tabela

    4. O table data tem que ser na mesma quantidade do cabeçalho, mesmo que sem texto, para não ocorrer quebra de dados

    • Cabeçalho: th (table header)
    • Linha: tr (table row)
    • Dados: td (table data)

    Linha 3 - Coluna 1 Linha 3 - Coluna 2 Linha 3 - Coluna 3
