Salvar

Sequência de comandos git:

  • git init
  • git remote (copiar link do novo repositório criado)
  • git add . (enviar novas alterações para o repositório)
  • git commit -m "" (comentário sobre alteração do repositório)
  • git push origin master/main/branch (envia os dados para o repositório)
  • git pull origin master (solicita os dados do repositório)
  • github pages - publica a página
